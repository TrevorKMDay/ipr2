# Inter-rater parent reliability

Trevor K.M. Day, Meghan Swanson, Jed T. Elison

Code: Trevor K.M. Day (trevor.day@georgetown.edu)

For more information and shared data, see the
[OSF project](https://osf.io/eh9dr/) (Day, Lamm & Elison, 2023).

## Overview

This project assesses the inter-rater reliability on social-communicative
behaviors of two-year-olds. We assess language skills with the MacArthur-Bates
Communicative Development Inventories (CDI; Fenson et al., 1994) and
social behaviors with the video-referenced ratings of
Reciprocal Social Behavior (vrRSB; Marrus et al., 2015) and rater social
competence with the Broad Autism Phenotype Questionnaire (BAPQ;
Hurley et al., 2007)

## Modeling

We implement a structural equation modeling approach (see the OSF project) to
assess what factors influence (1) how many words parents report their child
knows; and (2) how many social behaviors parents report their child displays.
